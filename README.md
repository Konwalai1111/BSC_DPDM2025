# Data Mining

> **Data Mining (การทำเหมืองข้อมูล) คือการสกัดรูปแบบหรือความรู้ที่น่าสนใจ ซึ่งเป็นข้อมูลที่ไม่ปรากฏซ้ำ ไม่เป็นที่ทราบมาก่อน และอาจนำไปใช้ประโยชน์ได้จากข้อมูลจำนวนมหาศาล**

เหตุผลที่ต้องมี Data Mining

> 1.การเติบโตอย่างรวดเร็วของข้อมูล

โลกมีการสร้างและจัดเก็บข้อมูลเพิ่มขึ้นอย่างมหาศาล (จากterabyteไปpetabyte) จากหลากหลายแหล่ง เช่น ธุรกรรมทางธุรกิจ, e-commerce, วิทยาศาสตร์, โซเชียลมีเดียและข้อมูลมัลติมีเดีย ซึ่งปริมาณข้อมูลนี้เกินกว่าขีดความสามารถของมนุษย์ที่จะวิเคราะห์และทำความเข้าใจด้วยวิธีการวิเคราะห์แบบเดิมได้
> 2.การขาดแคลนความรู้

เรากำลัง "จมอยู่ในข้อมูล แต่ขาดแคลนความรู้" แม้จะมีข้อมูลจำนวนมากแต่ความรู้หรือรูปแบบที่มีค่าและเป็นประโยชน์ เช่น แนวโน้มของลูกค้า, การทำนายความเสี่ยง, การค้นพบทางวิทยาศาสตร์ ยังคงถูกซ่อนอยู่ในข้อมูลเหล่านั้นและต้องการเครื่องมือเฉพาะทางในการดึงออกมา


ดังนั้น Data Mining จึงเป็นเครื่องมือในการเปลี่ยนชุดข้อมูลขนาดใหญ่ให้กลายเป็นความรู้ที่นำไปปฏิบัติได้ผ่านการวิเคราะห์ชุดข้อมูลโดยอัตโนมัติ
